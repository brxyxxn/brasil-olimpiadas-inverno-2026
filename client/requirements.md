## Packages
framer-motion | Smooth animations for the timeline and hero section
lucide-react | Icons for sports and UI elements
date-fns | Date formatting for the schedule

## Notes
The app displays Brazilian athletes for Milano Cortina 2026.
API endpoints:
- GET /api/athletes (returns athletes with their events)
- GET /api/events (returns all events flat list)
Images:
- Use Unsplash for winter sports backgrounds
- Use placeholders for athletes if no specific image is available, or generic sport icons
Colors:
- Brazilian flag colors: Green, Yellow, Blue
- Winter theme: White, Ice Blue, Dark Navy
